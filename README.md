# 🚀 Space Explorer Game (Java Swing)

## 📌 Description
A simple GUI-based Space Explorer Game built using Java Swing.  
The player controls a spaceship to avoid falling asteroids.

## 🎮 Features
- Spaceship movement (Left / Right)
- Falling asteroids
- Collision detection
- Game Over system

## 🛠 Concepts Used
✔ JFrame & JPanel  
✔ KeyListener (Keyboard control)  
✔ Timer (Game loop)  
✔ Collision Detection  
✔ OOP  

## 🎯 Objective
Avoid asteroids as long as possible and survive.

## ▶ How to Run
1. Open project in Eclipse / IntelliJ
2. Run main class
